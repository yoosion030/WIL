## 함수 인터페이스

// 함수의 스펙에 인터페이스를 활용

```ts
interface SumFunction {
  (a: number, b: number): number;
}
```

이렇게 함수 모양을 지정해줄 수 있다.

```ts
let test: SumFunction = (a: number, b: number): number => {
  return a + b;
};
```
